You can't park there !

Petit jeu 3D mignon et féerique développé avec Unity. Garer la voiture sans se planter — facile à dire !

Description

Dans You can't park there !, vous contrôlez une petite voiture dans des parkings farfelus. Obstacles, cônes, voitures... : garez-vous correctement pour gagner.

Caractéristiques

  • Contrôles simples : Z/S pour avancer/reculer, Q/D pour tourner.
  • Physique réaliste simplifiée (pas de rotation sur place).
  • Niveaux variés, ambiance mignonne et sonore féerique.
  • Modes WebGL et Build Windows (prévu).

Contrôles

Z / S Avancer / Reculer
Q / D Tourner
Echap Pause

Instructions (Windows)

Décompressez et lancez `YouCantParkThere.exe`. Si Windows bloque l’exécution, faites un clic droit → Exécuter en tant qu’administrateur ou débloquez via Propriétés → Débloquer.
