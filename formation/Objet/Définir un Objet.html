Définir un Objet

Qu'est-ce qu'un objet en programmation ?

Un objet est une unité fondamentale en Programmation Orientée Objet (POO). Il représente une entité du monde réel ou conceptuel, capable de stocker des données (appelées propriétés ou attributs) et de réaliser des actions (appelées méthodes). Un objet est une instance d'une classe, qui sert de modèle ou de plan pour créer plusieurs objets similaires.

Nommer un Objet

Un objet est une instance d'une classe, qui est un modèle ou un plan définissant les caractéristiques et les comportements que l'objet peut avoir. Lorsque nous créons un objet, nous devons lui donner un nom qui est généralement en rapport avec ce que représente l'objet. Par exemple, si nous créons un objet qui représente une horloge, nous pouvons le nommer Horloge, Time ou encore Montre. Le nom doit être significatif et décrire clairement ce que l'objet représente pour faciliter sa compréhension et son utilisation.

Attributs

Les attributs sont des caractéristiques ou des propriétés qui décrivent l'état d'un objet. Ils sont définis dans la classe et sont généralement représentés par des variables. Par exemple, dans notre classe Horloge, nous avons plusieurs attributs :

  • heure (de type int): représente l'heure actuelle.
  • minute (de type int): représente les minutes actuelles.
  • seconde (de type int): représente les secondes actuelles.
  • nom (de type String): représente le nom de l'horloge.

/** Exemple JAVA */
public class Horloge 
{
	public String nom = "Default" ;
	private int heure = 0 ;
	private int minute = 0 ;
	private int seconde = 0 ;
}

class Horloge:

	nom = "Default" 
	heure = 0
	minute = 0
	seconde = 0

Méthodes

Les méthodes sont des fonctions définies au sein d'une classe qui décrivent les comportements ou les actions que l'objet peut effectuer. Elles permettent de manipuler les attributs de l'objet et d'interagir avec celui-ci. Par exemple, dans notre classe Horloge, nous avons des méthodes comme :

  • setHeure: permet de définir l'heure de l'horloge.
  • setMinute: permet de définir les minutes de l'horloge, en prenant en compte la conversion des minutes excédentaires en heures.
  • setSeconde: permet de définir les secondes de l'horloge, avec la conversion des secondes excédentaires en minutes.
  • afficherHeure: permet d'afficher l'heure au format hh:mm:ss


/** Exemple JAVA */
public class Horloge 
{
	public String nom = "Default" ;
	private int heure = 0 ;
	private int minute = 0 ;
	private int seconde = 0 ;

	public void setHeure(int heure) {
        if (heure >= 0) {
            this.heure = heure;
        } else {
            System.out.println("Heure invalide.");
        }
    }

    public void setMinute(int minute) {
        if (minute >= 0) {
            this.heure += minute / 60;          // Ajoute les heures supplémentaires si minute >= 60
            this.minute = minute % 60;           // Garde les minutes restantes (0-59)
            setHeure(this.heure);                // Mets à jour l'heure au cas où elle dépasserait 24
        } else {
            System.out.println("Minute invalide.");
        }
    }

    public void setSeconde(int seconde) {
        if (seconde >= 0) {
            this.minute += seconde / 60;         // Ajoute les minutes supplémentaires si seconde >= 60
            this.seconde = seconde % 60;         // Garde les secondes restantes (0-59)
            setMinute(this.minute);              // Mets à jour les minutes et potentiellement l'heure
        } else {
            System.out.println("Seconde invalide.");
        }
    }

    public void setTime(int heure, int minute, int seconde) {
        setHeure(heure);
        setMinute(minute);
        setSeconde(seconde);
    }

    public String afficherHeure() {
        return String.format("%02d:%02d:%02d", heure, minute, seconde);
    }

}

class Horloge:

	def set_heure(self, heure):

		self.heure = heure
	else:
		print("Heure invalide.")

	def set_minute(self, minute):
		if minute >= 0:
			self.heure += minute // 60          # Ajoute les heures supplémentaires si minute >= 60
			self.minute = minute % 60           # Garde les minutes restantes (0-59)
			self.set_heure(self.heure)          # Mets à jour l'heure si nécessaire
		else:
			print("Minute invalide.")

	def set_seconde(self, seconde):
		if seconde >= 0:
			self.minute += seconde // 60        # Ajoute les minutes supplémentaires si seconde >= 60
			self.seconde = seconde % 60         # Garde les secondes restantes (0-59)
			self.set_minute(self.minute)        # Mets à jour les minutes et potentiellement l'heure
		else:
			print("Seconde invalide.")

	def set_time(self, heure, minute, seconde):
		self.set_heure(heure)
		self.set_minute(minute)
		self.set_seconde(seconde)

	def afficher_heure(self):
		return f"{self.heure:02d}:{self.minute:02d}:{self.seconde:02d}"

Constructeur

Un constructeur est une méthode spéciale qui est appelée lors de la création d'un nouvel objet. Il a le même nom que la classe et est utilisé pour initialiser les attributs de l'objet. Dans notre classe Horloge, nous avons un constructeur qui prend les valeurs d'heure, de minute et de seconde comme paramètres et les utilise pour initialiser l'état de l'objet. Par exemple : 


/** Exemple JAVA */
public class Horloge 
{
	public Horloge() {}

	public Horloge(String nom) {
        this.nom = nom;
    }

    public Horloge(String nom, int heure, int minute, int seconde) {
        this.nom = nom;
        setTime(heure, minute, seconde);
    }

}


# Exemple PYTHON
class Horloge:

	def __init__(self, nom, heure=0, minute=0, seconde=0):
	self.nom = nom
	self.set_time(heure, minute, seconde)

Exemple d'utilisation

Voici un exemple d'utilisation de l'objet Horloge, incluant la création d'une instance avec le constructeur, l'utilisation de la méthode setMinute et l'affichage de l'état de l'horloge : 


/** Exemple JAVA */
public class Main {
    public static void main(String[] args) {
        // Création de l'objet Horloge avec le constructeur
        Horloge maHorloge = new Horloge("Ma Horloge", 10, 45, 30);

        // Utilisation de la méthode setMinute pour modifier les minutes
        maHorloge.setMinute(75);  // Cela ajoutera 1 heure et mettra les minutes à 15

        // Affichage de l'état de l'horloge
        System.out.println(horloge.nom + " : " + horloge.afficherHeure());
    }
}
