Histoire

avaScript aurait été créé en dix jours à peine par Brendan Eich pour le compte de Netscape en 1995. Le langage a porté successivement les noms Mocha puis LiveScript et enfin JavaScript.

À sa création, JavaScript avait une syntaxe un peu proche de celle du Java, des fonctions de première classe comme on en trouve dans le langage Scheme, des types dynamiques comme Lisp et des prototypes comme ceux présents dans le langage Self.

Destiné à fournir un langage de script simple au navigateur Netscape Navigator 2, il a, à l'époque, longtemps été critiqué pour son manque de sécurité, son développement peu poussé et l'absence de messages d'erreur explicites rendant dure son utilisation.

Le 4 décembre 1995, suite à une association avec le constructeur Sun, Netscape rebaptise son langage Javascript (un clin d’œil au langage Java développé par Sun).

Introduction

JavaScript (JS) est un langage de programmation principalement utilisé pour le développement web. Contrairement à des langages comme Java qui sont compilés, JavaScript est un langage interprété qui s'exécute directement dans le navigateur du client. Il permet de rendre les pages web interactives en manipulant le HTML et le CSS dynamiquement.

avaScript a été créé en 1995 par Brendan Eich chez Netscape. Initialement appelé LiveScript, il a été renommé JavaScript pour des raisons marketing. Aujourd'hui, c'est l'un des langages les plus populaires et il est supporté par tous les navigateurs modernes.

Pourquoi utiliser JavaScript ?

JavaScript permet d'ajouter de l'interactivité aux pages web, comme des animations, des mises à jour dynamiques du contenu ou encore des formulaires interactifs. Il est utilisé aussi bien côté client que côté serveur avec des technologies comme Node.js, ce qui en fait un langage très polyvalent.

Pour commencer à coder en JavaScript, il suffit d'avoir un simple bloc-notes et un navigateur web moderne comme Chrome, Firefox ou Edge. Les navigateurs disposent déjà d'une console de développement permettant d'exécuter du code JavaScript sans installer de logiciels supplémentaires. Bien que des outils plus avancés existent, un environnement minimaliste suffit pour débuter et comprendre les bases du langage.

Tous les navigateurs sont fournis avec un moteur JavaScript. C’est un logiciel qui permet d’exécuter du code JavaScript directement depuis le navigateur.
