Contrôlez le Déroulement d'un Programme avec des Conditions

Les conditions permettent de contrôler la logique d’un programme en fonction de certaines situations ou valeurs. Grâce à elles, vous pouvez exécuter différentes parties de code selon les données disponibles ou les choix effectués par l'utilisateur.

En Java, il existe deux principales structures conditionnelles : if/else et switch, qui permettent de gérer les décisions avec souplesse et efficacité.

La Structure if et else

La structure conditionnelle if permet d’exécuter un bloc de code si une condition est vraie. Si la condition est fausse, un autre bloc de code peut être exécuté grâce à else. 


    public static void main(String[] args) {
        int age = 18;
        if (age >= 18) 
        {
            System.out.println("Vous êtes majeur.");
        } 
        else 
        {
            System.out.println("Vous êtes mineur.");
        }
    }
  • Si la valeur de age est supérieure ou égale à 18, le programme affiche "Vous êtes majeur."
  • Sinon, il affiche "Vous êtes mineur."

La combinaison de if, else if, et else permet de gérer plusieurs conditions dans un programme. 


    if (condition1) {
        // Code à exécuter si condition1 est vraie
    } else if (condition2) {
        // Code à exécuter si condition1 est fausse et condition2 est vraie
    } else if (condition3) {
        // Code à exécuter si condition1 et condition2 sont fausses, mais condition3 est vraie
    } else {
        // Code à exécuter si toutes les conditions précédentes sont fausses
    }

Évaluation des Résultats d'un Étudiant

  • Si le score est supérieur ou égal à 90, le programme affiche "Excellent : A".
  • Si le score est entre 80 et 89, il affiche "Très bien : B".
  • Si le score est entre 70 et 79, il affiche "Bien : C", et ainsi de suite.
  • Si aucune des conditions n’est remplie (par exemple un score inférieur à 60), le programme exécute le bloc else et affiche "Échec : F".

    public static void main(String[] args) {
        int score = 75;
    
        if (score >= 90) {
            System.out.println("Excellent : A");
        } else if (score >= 80) {
            System.out.println("Très bien : B");
        } else if (score >= 70) {
            System.out.println("Bien : C");
        } else if (score >= 60) {
            System.out.println("Passable : D");
        } else {
            System.out.println("Échec : F");
        }
    }

La Structure switch

La structure switch est utile lorsqu'il y a plusieurs cas possibles basés sur une seule valeur. Elle simplifie la lecture du code par rapport à une série de if/else if. 


    int jour = 3;
    
    switch (jour) {
        case 1:
            System.out.println("Lundi");
            break;
        case 2:
            System.out.println("Mardi");
            break;
        case 3:
            System.out.println("Mercredi");
            break;
        default:
            System.out.println("Jour inconnu");
        }
  • La valeur de la variable jour est comparée aux cas (case).
  • Si jour vaut 1, le programme affiche "Lundi", et ainsi de suite.
  • Le mot-clé default définit ce qui est exécuté si aucun des cas ne correspond.
  • Le mot-clé break est utilisé pour arrêter l’exécution après chaque cas.

Évaluation des Résultats d'un Étudiant

Écrivez un programme qui affiche un message personnalisé en fonction d'une note scolaire (A, B, C, D ou F). 


    char note = 'B';
    
    switch (note) 
    {
        case 'A':
            System.out.println("Excellent !");
            break;
        case 'B':
            System.out.println("Très bien !");
            break;
        case 'C':
            System.out.println("Bien.");
            break;
        case 'D':
            System.out.println("Passable.");
            break;
        case 'F':
            System.out.println("Échec.");
            break;
        default:
            System.out.println("Note invalide.");
    }
