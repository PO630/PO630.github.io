Écrivez une Fonction

En programmation, une fonction (ou méthode en Java) est un bloc de code qui exécute une tâche spécifique. Elle peut prendre des paramètres en entrée, effectuer des opérations, et retourner une valeur ou simplement exécuter un traitement. 


    public class Fonctions {
        public static void main(String[] args) {
            saluer("François-Xavier");
            int somme = addition(5, 3);
            System.out.println("Somme : " + somme);
        }
    
        public static void saluer(String nom) {
            System.out.println("Bonjour, " + nom + " !");
        }
    
        public static int addition(int a, int b) {
            return a + b;
        }
    }

Rôle des fonctions

  • Réutiliser du code : Éviter la duplication de code en définissant une tâche commune.
  • Organiser le programme : Diviser un programme en unités logiques plus simples à comprendre.
  • Faciliter la maintenance : Modifier une fonction modifie son comportement partout où elle est utilisée.

Déclaration d'une fonction

En Java, une fonction (appelée méthode) est déclarée dans une classe.


    modificateurDeVisibilité typeRetour nomFonction(paramètres) {
        // Corps de la fonction
        return valeur; // (optionnel, si le typeRetour n'est pas void)
    }
  • Modificateur de visibilité : Définit où la fonction peut être utilisée (public, private, etc.).
  • Type de retour : Type de la valeur retournée (ou void si aucune valeur n'est retournée).
  • Nom de la fonction : Identifiant descriptif pour appeler la fonction.
  • Paramètres : Valeurs passées à la fonction (optionnel).

Exemples



    // Une fonction qui retourne une valeur
    public int addition(int a, int b) {
        return a + b;  // Retourne la somme de a et b
    }

    //Une fonction qui ne retourne rien (void)
    public void afficherMessage(String message) {
        System.out.println(message);
    }    

    // Appel de la fonction 
    int resultat = addition(5, 3); // resultat vaut 8
    afficherMessage("Bonjour, monde !");

Surcharge des fonctions

Java permet d'avoir plusieurs fonctions avec le même nom mais des paramètres différents. C'est ce qu'on appelle la surcharge. 



    public int addition(int a, int b) {
        return a + b;
    }
    
    public double addition(double a, double b) {
        return a + b;
    }
    
    public int addition(int a, int b, int c) {
        return a + b + c;
    }
    
    int x = addition(5, 3);          // Appelle la première version
    double y = addition(2.5, 3.8);   // Appelle la deuxième version
    int z = addition(1, 2, 3);       // Appelle la troisième version

Portée des fonctions

    Une fonction peut être :
  • Public : Accessible de n'importe où.
  • Private : Accessible uniquement dans la classe où elle est définie.
  • Protected : Accessible dans le même package ou par des classes héritées.
