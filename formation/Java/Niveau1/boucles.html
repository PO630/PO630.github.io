Écrivez une Boucle dans vos Fonctions

En programmation, il est courant de devoir exécuter plusieurs fois un ensemble d'instructions, que ce soit pour parcourir une collection, traiter des données ou résoudre des problèmes mathématiques. Pour répondre à ce besoin, les langages de programmation proposent des structures permettant d'effectuer des itérations : les boucles et la récursion.

  • Les boucles permettent d'exécuter un bloc de code plusieurs fois en fonction d'une condition.
  • La récursion, quant à elle, repose sur une méthode qui s'appelle elle-même pour résoudre un problème en le divisant en sous-problèmes.

La boucle for

La boucle for est une structure itérative utilisée lorsque le nombre d'itérations est connu à l'avance. Elle est particulièrement adaptée pour parcourir des collections (comme des tableaux ou des listes) ou pour effectuer des tâches répétées sur une plage de valeurs. 


    // Afficher les nombres de 1 à 5 :
    for (int i = 1; i <= 5; i++) {
        System.out.println(i);
    }
  • Initialisation "(int i = 1;)" : Exécutée une seule fois avant le début de la boucle, elle initialise une variable.
  • Condition "(i <= 5;)" : Testée avant chaque itération, elle détermine si la boucle doit continuer.
  • Mise à jour "(i++)" : Exécutée après chaque itération, elle modifie la variable pour préparer la prochaine exécution.

La boucle while

La boucle while est utilisée lorsque le nombre d'itérations est inconnu à l'avance. Elle exécute un bloc de code tant qu'une condition donnée reste vraie. Si la condition est initialement fausse, la boucle ne s'exécute jamais. 


    // Afficher les nombres de 1 à 5 :
    int i = 1;
    while (i <= 5) {
        System.out.println(i);
        i++; // Mise à jour
    }
  • Condition "(i <= 5)" : Testée avant chaque itération, elle détermine si la boucle doit continuer ou s'arrêter.

Le do-while est une structure de boucle en Java qui permet d'exécuter un bloc de code au moins une fois, puis de le répéter tant qu'une condition est vraie. 


    do {
        // Instructions à exécuter
    } while (condition);
  • do : C'est le début de la boucle. Le code à l'intérieur du bloc do sera exécuté au moins une fois, même si la condition est fausse dès le départ.
  • while (condition) : Après avoir exécuté les instructions du bloc do, la condition est vérifiée. Si la condition est vraie, la boucle recommence, sinon elle s'arrête.

La récursion

La récursion est une approche dans laquelle une méthode s'appelle elle-même pour résoudre un problème en le décomposant en sous-problèmes plus simples. La récursion repose sur deux éléments essentiels :

  • Une condition d'arrêt qui termine la chaîne d'appels.
  • Un appel récursif qui rapproche le problème de la condition d'arrêt.
Bien que puissante, la récursion peut consommer davantage de mémoire en raison de l'empilement des appels sur la pile d'exécution. 


    // Calcul du facteuriel d'un nombre :
    public static int factoriel(int n) {
        if (n == 0) { // Condition d'arrêt
            return 1;
        }
        return n * factoriel(n - 1); // Appel récursif
    }

En résumé, les boucles et la récursion sont des outils essentiels pour résoudre des problèmes répétitifs. Le choix de l'une ou l'autre dépend du problème à résoudre et des contraintes de performance ou de lisibilité du code. Passons maintenant à des exemples pratiques pour comprendre leur fonctionnement en détail.

    for (int i = 2; i <= 20; i += 2) 
    { // Incrémentation de 2 pour les nombres pairs
        System.out.println(i);
    }

Écrivez une fonction récursive qui calcule la somme des nombres entiers de 1 à n. Par exemple, pour n = 5, la somme est 1 + 2 + 3 + 4 + 5 = 15. 


    public static int sommeRecursive(int n) {
        if (n == 1) { // Condition d'arrêt
            return 1;
        }
        return n + sommeRecursive(n - 1); // Appel récursif
    }

import java.util.Random; // Pour générer des nombres aléatoires

public class Main

    public static void main(String[] args) 
    {
        Random random = new Random(); // Création d'un générateur aléatoire

        System.out.println("Nombre aléatoire généré : " + nombreAleatoire);

        int nombre = 1; // Initialisation de la variable

        while (nombre <= nombreAleatoire) { // Condition pour continuer la boucle
            int carre = nombre * nombre; // Calcul du carré
            System.out.println(nombre + " x " + nombre + " = " + carre); // Affichage du résultat
            nombre++; // Incrémentation de la variable
        }
